# Ashvik Buildcon Website

A modern, responsive real estate website for **Ashvik Buildcon**, showcasing premium 2BHK, 3BHK, and 4BHK flats in prime Delhi locations like Mukherjee Nagar, GTB Nagar, and Model Town.

---

## 🚀 Overview

This website is designed to present Ashvik Buildcon as a trusted real estate developer with over a decade of experience. It focuses on showcasing properties, building trust, and generating leads through a clean and professional interface.

---

## ✨ Features

* Fully responsive design (mobile-first)
* Modern real estate UI
* Property showcase section
* Customer testimonials
* Animated stats counters
* Contact & inquiry form with validation
* Smooth scrolling and interactive elements

---

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS (CDN)
* Custom CSS
* Vanilla JavaScript
